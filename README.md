# T20 World Cup Analytics Dashboard

The T20 World Cup Analytics Dashboard is a data analytics project developed to analyze international T20 match data using interactive visualizations and performance metrics.

The dashboard transforms ball-by-ball cricket data into structured insights across players, teams, matches, and venues to support deeper performance analysis.

Project Objective

Cricket match data is large, detailed, and difficult to interpret in raw form.

This project aims to:
	•	Analyze player performance across tournaments
	•	Compare team performance metrics
	•	Identify scoring and wicket trends over years
	•	Evaluate venue impact on match outcomes
	•	Present insights using an interactive dashboard

⸻

Problem Statement
	•	Ball-by-ball data is complex and unstructured
	•	Hard to manually identify performance trends
	•	No centralized view for player, team, and venue insights
	•	Difficult to compare multi-year tournament data

⸻

Proposed Solution

The dashboard provides:
	•	Tournament-level summary insights
	•	Match-wise performance breakdown
	•	Batting and bowling analytics
	•	Team comparison metrics
	•	Venue-based performance analysis
	•	Interactive filtering for deeper exploration

⸻

Dashboard Sections

1. Overview Hub
	•	Total runs, matches, wickets, overs
	•	Year-wise scoring trends
	•	Top performers

2. Match Summary
	•	Venue, year, toss decision
	•	Innings order and winner
	•	High-scoring and close-match identification

3. Batting Analytics
	•	Top run scorers
	•	Strike rate comparison
	•	Boundaries contribution
	•	Over-wise scoring trends
	•	Death over (16–20) performance analysis

4. Bowling Analytics
	•	Leading wicket takers
	•	Economy rate comparison
	•	Wickets by over
	•	Dismissal type distribution

5. Team Comparison
	•	Runs scored vs runs conceded
	•	Win percentage
	•	Head-to-head insights
	•	Performance consistency

6. Venue Analysis
	•	Average runs by venue
	•	Average wickets by venue
	•	Batting-friendly vs bowler-friendly grounds
	•	Match winner distribution

Dataset

Source: Kaggle – Ball-by-Ball IT20 Dataset
	•	1,800+ international T20 matches
	•	20+ teams
	•	Multi-year tournament coverage
	•	Ball-level detailed match records

The dataset was cleaned, structured, and transformed before visualization.

Key Findings
	•	Scoring rates have increased across tournaments
	•	Death overs (16–20) are critical for match outcomes
	•	A small group of elite players significantly impact results
	•	Balanced teams with strong bowling depth perform consistently
	•	Venue conditions influence scoring and wicket-taking patterns
	•	Toss decisions are less influential than venue and innings order
