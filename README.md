🏏 T-20 World Cup Analytics Dashboard

An interactive data analytics dashboard built to explore performance trends, player statistics, team comparisons, and venue insights across multiple T-20 World Cups.

This project transforms large, scattered ball-by-ball cricket data into meaningful visual insights for analysts, cricket enthusiasts, and decision-makers.

⸻

📌 Project Overview

The T-20 World Cup Analytics Dashboard integrates match data, batting performance, bowling analysis, team statistics, and venue behavior into a single interactive platform.

The objective of this project is to:
	•	Summarize large-scale cricket data
	•	Identify performance trends across tournaments
	•	Analyze player impact and team consistency
	•	Evaluate venue influence on match outcomes

⸻

📂 Dataset Information

Source: Kaggle
Dataset: Ball-by-Ball IT20 Dataset

Dataset Coverage:
	•	1,800+ International T-20 Matches
	•	20+ Participating Teams
	•	Multi-year World Cup editions
	•	Ball-by-ball level data
	•	Match summaries and venue details

Key Fields Used:
	•	Runs, wickets, overs
	•	Strike rate, economy rate
	•	Dismissal types
	•	Toss decisions
	•	Innings information
	•	Venue statistics

⸻

📊 Dashboard Sections & Insights

🏠 1. Overview Hub
	•	Total runs, matches, wickets, overs
	•	Year-wise scoring trends
	•	Top batters and bowlers
	•	Tournament-level performance patterns

Insight: Scoring has increased over the years, reflecting aggressive modern batting styles.

⸻

📝 2. Match Summary
	•	Year, venue, toss decision, innings order
	•	Match winner analysis
	•	High-scoring and close-match identification

Insight: Venue and innings order impact results more than toss decisions.

⸻

🏏 3. Batting Analytics
	•	Top run scorers
	•	Strike rate comparison
	•	Boundaries contribution
	•	Over-wise scoring trends
	•	Scatter relationship: Balls Faced vs Runs vs Strike Rate

Key Finding: Overs 16–20 (death overs) generate the highest scoring rates.

⸻

🎯 4. Bowling Analytics
	•	Leading wicket-takers
	•	Economy rate comparison
	•	Wicket distribution by over
	•	Dismissal type analysis

Key Finding:
	•	Early and middle overs show peak wicket-taking.
	•	“Caught” is the most common dismissal type.

⸻

🤝 5. Team Comparison
	•	Runs scored vs runs conceded
	•	Win percentage
	•	Head-to-head matrix
	•	Team consistency across venues

Insight: Balanced teams with strong bowling depth show higher win consistency.

⸻

🏟 6. Venue Analysis
	•	Average runs per venue
	•	Average wickets per venue
	•	Batting-friendly vs bowler-friendly grounds
	•	Venue-wise winner distribution

Insight: Venue characteristics significantly influence match outcomes and scoring patterns.

⸻

🔎 Major Findings
	•	Scoring trends are increasing across tournaments.
	•	A small group of elite players heavily influence match results.
	•	Death overs (16–20) are critical in both batting and bowling.
	•	Teams with deeper bowling attacks perform more consistently.
	•	Toss decisions influence strategy but are not primary result drivers.
	•	Venue conditions strongly shape match dynamics.

⸻

🎯 Recommendations
	•	Improve death-over strategies in both batting and bowling.
	•	Select teams based on venue-specific pitch conditions.
	•	Focus on consistent performers for tournament planning.
	•	Use over-wise insights to refine powerplay and middle-over tactics.
	•	Future enhancement: Implement predictive modeling for match outcome forecasting.
